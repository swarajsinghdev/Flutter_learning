# Lesson 4 — Tasks & navigation

Mini task app: **home list**, **add task screen**, data returned with **`Navigator.pop`**, **`ListView.builder`**, **`setState`**, **`TextEditingController`**.

## Layout

```
lib/
├── main.dart
├── models/
│   └── task.dart
├── screens/
│   ├── home_screen.dart
│   └── add_task_screen.dart
└── widgets/
    ├── task_list_tile.dart
    └── empty_tasks_placeholder.dart
```

Run: `flutter run` from this folder.
